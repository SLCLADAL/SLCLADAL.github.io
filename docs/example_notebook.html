1 Introduction

This sample notebook uses R markdown. One of the main features of Markdown is its ability to display code snippets and their corresponding output in a readable format as shown below.

sentence <- "This is an example sentence."

We can now split the example sentence and count the number of tokens it consists of.

length(unlist(strsplit(sentence, " ")))
## [1] 5

The example sentnece consists of 5 tokens.
